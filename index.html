This map of Rügen island was manually created using the sand tileset by William.Thompsonj, Daniel Eddeland from OpenGameArt.Org.

  • Rügen Beach

The following maps of selected islands where created automatically from OpenStreetMap data, © OpenStreetMap contributors. The scripts can be found at Github. Ideas for the future development include an extended sand tileset, an island hopping map for navigation, treatment of archipelagos, addition of further features to the map, for example settlements, and improved coastline generalization.
